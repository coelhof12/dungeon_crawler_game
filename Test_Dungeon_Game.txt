1. Navigate to Project Directory:

cd ~/Code/ProjectDevelopment/dungeon_crawler_game/src


2. Compile the Java Files:

javac dungeoncrawler/*.java


3. Create a JAR File:

jar cfm ../dungeoncrawler.jar ../MANIFEST.MF dungeoncrawler/*.class


4. Run the JAR File:\

java -jar ../dungeoncrawler.jar
